# Business Process UML Diagrams

This repository contains my learning notes and UML diagrams created to understand real-world business processes used in OMS / Retail / eCommerce systems.

The goal of this repo is not just diagram creation, but to build clarity on how business flows work from order creation to fulfillment, shipping, inventory, and status changes.

## What I’m Learning

- How business processes move step-by-step  
- How different systems interact (OMS, ERP, WMS, eCommerce, POS)  
- Order lifecycle and fulfillment lifecycle  
- Inventory movement and allocation logic  
- Converting real processes into UML flow diagrams  

## Why This Repo Exists

As a backend learner, I want to understand that software is not only code — it runs business operations.  
These diagrams help me connect business logic with system design.

## Current Focus

Learning business processes deeply first, then mapping them to:

- Database entities  
- APIs  
- Services  
- Architecture decisions
